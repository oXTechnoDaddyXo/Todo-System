# README - Western Railway Dispatch

========================
🇩🇪 DEUTSCH
========================

🚂 Western Railway Dispatch System (VORP / RedM)

Ein Railroad Dispatch Script für RedM mit Discord Webhook Integration.

FEATURES:
- NUI Dispatch Interface
- Bahnhof Auswahl
- Ankunft / Abfahrt / Annäherung
- Discord Webhook (farbig)
- VORP Job System (railroad)
- Ingame Notifications

INSTALLATION:
1. Resource in resources/ legen
2. ensure khalis_railroad_announce in server.cfg
3. Job: railroad setzen
4. Webhook in config.lua eintragen

COMMAND:
 /railannounce

BENUTZUNG:
- Bahnhof auswählen
- Typ wählen
- Zeit optional
- Senden klicken

========================
🇬🇧 ENGLISH
========================

🚂 Western Railway Dispatch System (VORP / RedM)

A railroad dispatch system for RedM with Discord webhook integration.

FEATURES:
- NUI Dispatch interface
- Station selection
- Arrival / Departure / Approaching
- Discord webhook (colored embeds)
- VORP job system (railroad)
- In-game notifications

INSTALLATION:
1. Place resource into resources/
2. Add ensure khalis_railroad_announce to server.cfg
3. Set job: railroad
4. Add webhook in config.lua

COMMAND:
 /railannounce

USAGE:
- Select station
- Choose type
- Optional time
- Press send
