## 🪶 VORP Frontier Clipboard (Todo System)

------------------------------------------------------------------------

## 🇩🇪 Deutsch

### 📜 Beschreibung

Das **VORP Frontier Clipboard** ist ein immersives To-Do-System im
Western-Stil für RedM Server.\
Spieler können ihre Notizen und Aufgaben direkt im Spiel verwalten --
stilvoll wie auf einem echten Klemmbrett im Sheriff-Office.

### ✨ Features

-   Öffnen per Command (/todo)
-   Aufgaben hinzufügen, abhaken und löschen
-   Automatisches Speichern (Crash-/Restart-sicher)
-   Immersives Western UI (Klemmbrett / Papier Stil)
-   Mehrsprachig (Deutsch & Englisch)
-   Leichtgewichtig & performant
-   Stabil und serverfreundlich

### 🎮 Verwendung

/todo

### ⚙️ Installation

1.  Resource in den resources Ordner legen
2.  ensure vorp_todo in server.cfg
3.  Server starten

### 🧠 Abhängigkeiten

-   VORP Core

### 🛠️ Konfiguration

Config.Locale = "de"
